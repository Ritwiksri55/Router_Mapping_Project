# WiFi / IoT Deployment Planner

## Overview
This project is a **WiFi / IoT router deployment planner** that allows you to model physical spaces and simulate dynamic network hardware placement. You can drag, drop, and auto-deploy router nodes to analyze coverage and signal quality.

It is built using **HTML, CSS, JavaScript**, and uses the **Leaflet** library for map rendering and **p5.js** for additional graphical simulations.

---

## Features
- Pan / move across the canvas  
- Draw custom regions for router deployment  
- Drop router nodes manually or auto-deploy to selected areas  
- Adjust operating frequency (2.4GHz or 5GHz)  
- View coverage statistics: Optimal, Weak, Interference, Dead Zones  
- Interactive color-coded legend  
- Responsive UI with modern styling  

---

## How to Use
1. Open the project online via GitHub Pages:  
   [Live Demo](https://your-username.github.io/router-mapping/)  

2. Drag and drop routers or use auto-deploy  
3. Adjust sliders and frequency to see how coverage changes  
4. Monitor coverage statistics in real-time

## Technologies Used
- **HTML / CSS / JavaScript** – Core project  
- **Leaflet.js** – Map visualization  
- **p5.js** – Graphics and simulation
